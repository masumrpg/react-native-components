## Core Components

### Theme Provider

The `RNCProvider` is the root component that provides theme context to your entire application:

```tsx
import { RNCProvider } from 'rnc-theme';

export default function App() {
  return (
    <RNCProvider defaultTheme="system">
      {/* Your app content */}
    </RNCProvider>
  );
}
```

### Theme Hooks

Two primary hooks for accessing and using themes:

<Tabs>
  <TabItem label="useTheme">
    ```tsx
    import { useTheme } from 'rnc-theme';

    function MyComponent() {
      const { theme, mode, setMode, updateCustomTheme } = useTheme();

      return (
        <View style={{ backgroundColor: theme.colors.background }}>
          <Text style={{ color: theme.colors.text }}>Hello World</Text>
        </View>
      );
    }
    ```
  </TabItem>
  <TabItem label="useThemedStyles">
    ```tsx
    import { useThemedStyles } from 'rnc-theme';

    function MyComponent() {
      const styles = useThemedStyles(createStyles);

      return <View style={styles.container} />;
    }

    const createStyles = (theme: Theme) => ({
      container: {
        backgroundColor: theme.colors.background,
        padding: theme.spacing.md,
        borderRadius: theme.components.borderRadius.md,
      },
    });
    ```
  </TabItem>
</Tabs>

### Theme Registry

Manage multiple theme presets with the theme registry:

```tsx
import { themeRegistry } from 'rnc-theme';

// Register a custom theme
themeRegistry.register('myTheme', {
  colors: {
    primary: '#FF6B6B',
    secondary: '#4ECDC4',
    // ... other colors
  },
  // ... other theme properties
});

// Apply the theme
const { setActivePreset } = useTheme();
setActivePreset('myTheme');
```

## Theme Modes

Supports three theme modes:

- **Light Mode**: Optimized for bright environments
- **Dark Mode**: Optimized for low-light environments
- **System Mode**: Automatically follows device settings

```tsx
const { mode, setMode } = useTheme();

// Switch to dark mode
setMode('dark');

// Follow system preference
setMode('system');
```

## Performance Optimizations

### Memoization
All theme calculations are memoized to prevent unnecessary re-renders:

```tsx
const styles = useThemedStyles(createStyles); // Memoized automatically
```

### Lazy Loading
Theme presets are loaded on-demand to reduce initial bundle size.

### Efficient Updates
Only components using changed theme properties re-render during theme switches.

## Accessibility

RNC Theme includes built-in accessibility features:

- **High Contrast Support**: Automatic contrast adjustments
- **Reduced Motion**: Respects system motion preferences
- **Screen Reader Support**: Semantic color naming
- **Focus Indicators**: Clear focus states for all interactive elements

## Integration Examples

### Basic Setup

```tsx
import React from 'react';
import { RNCProvider } from 'rnc-theme';
import { NavigationContainer } from '@react-navigation/native';

export default function App() {
  return (
    <RNCProvider
      defaultTheme="system"
      persistTheme={true}
    >
      <NavigationContainer>
        {/* Your navigation structure */}
      </NavigationContainer>
    </RNCProvider>
  );
}
```

### Custom Theme Configuration

```tsx
const customThemeConfig = {
  colors: {
    primary: '#6366F1',
    secondary: '#8B5CF6',
    background: '#F8FAFC',
    surface: '#FFFFFF',
    // ... other colors
  },
  components: {
    borderRadius: {
      xs: 4,
      sm: 6,
      md: 8,
      lg: 12,
      xl: 16,
      full: 9999,
    },
  },
};

<RNCProvider customTheme={customThemeConfig}>
  {/* Your app */}
</RNCProvider>
```
