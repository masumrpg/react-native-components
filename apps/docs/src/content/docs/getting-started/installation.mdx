## Essential Configuration

### Babel Configuration

<Aside type="caution" title="Important">
  The Reanimated plugin **must** be the last item in your plugins array.
</Aside>

```javascript title="babel.config.js"
module.exports = {
  presets: ['module:metro-react-native-babel-preset'],
  plugins: [
    // Other plugins...
    'react-native-reanimated/plugin', // ⚠️ Must be last!
  ],
};
```

### TypeScript Configuration

RNC Theme includes comprehensive TypeScript definitions. Update your `tsconfig.json`:
<Aside type="tip">
  Add the following to your `tsconfig.json` if you haven't already and using typeScript.
</Aside>

```json title="tsconfig.json"
{
  "compilerOptions": {
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "moduleResolution": "node",
    "allowSyntheticDefaultImports": true
  },
  "include": [
    "**/*.ts",
    "**/*.tsx"
  ]
}
```

    import { Stack, usePathname } from 'expo-router';
