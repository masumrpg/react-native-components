## üöÄ Getting Started

To start using these components in your project:

1. **Install the package**:
   ```bash
   npm install rnc-theme
   # or
   yarn add rnc-theme
   ```

2. **Setup the theme provider**:
   ```tsx
   import { ThemeProvider } from 'rnc-theme';

   function App() {
     return (
       <ThemeProvider>
         {/* Your app content */}
       </ThemeProvider>
     );
   }
   ```

3. **Import and use components**:
   ```tsx
   import { Button, VStack, Box, Text } from 'rnc-theme';

   function MyScreen() {
     return (
       <VStack spacing="md" padding="lg">
         <Box variant="card">
           <Text>Welcome to RNC Theme!</Text>
         </Box>
         <Button onPress={() => console.log('Pressed!')}>
           Get Started
         </Button>
       </VStack>
     );
   }
   ```
