Layout components provide flexible and powerful tools for organizing and structuring your React Native application's UI. These components are built with theme support and offer consistent spacing, alignment, and styling options.

## Available Components

- **HStack** - Horizontal stack layout
- **VStack** - Vertical stack layout
- **ZStack** - Absolute positioned stack
- **Grid** - Responsive grid layout
- **Center** - Centers content both horizontally and vertically
- **Box** - Themed container with multiple variants

## HStack

Arranges children horizontally in a row with flexible spacing and alignment options.

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `spacing` | `SpacingKey` | - | Space between children |
| `align` | `FlexAlignType` | `'stretch'` | Cross-axis alignment |
| `justify` | `FlexJustifyType` | `'flex-start'` | Main-axis alignment |
| `wrap` | `boolean` | `false` | Allow children to wrap |
| `padding` | `SpacingKey` | - | Internal padding |
| `margin` | `SpacingKey` | - | External margin |
| `backgroundColor` | `ColorKey` | - | Background color |
| `borderRadius` | `BorderRadiusKey` | - | Border radius |
| `themed` | `boolean` | `false` | Apply theme background |

### Example

<Tabs>
<TabItem label="Basic Usage">

```tsx

  return (
    <HStack spacing="md" justify="space-between" padding="lg">
```

## ZStack

Positions children absolutely, allowing them to overlap. Useful for creating layered interfaces.

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `padding` | `SpacingKey` | - | Internal padding |
| `margin` | `SpacingKey` | - | External margin |
| `backgroundColor` | `ColorKey` | - | Background color |
| `borderRadius` | `BorderRadiusKey` | - | Border radius |
| `themed` | `boolean` | `false` | Apply theme background |

### Example

```tsx
import { ZStack, Box, Text } from 'rnc-theme';

function OverlayLayout() {
  return (
    <ZStack height={200} backgroundColor="surface" borderRadius="lg">
      <Box
        position="absolute"
        top={0}
        left={0}
        right={0}
        bottom={0}
        backgroundColor="primary"
        borderRadius="lg"
      />
      <Box
        position="absolute"
        top={20}
        left={20}
        padding="md"
        backgroundColor="white"
        borderRadius="md"
      >
        <Text style={{ color: 'black', fontWeight: 'bold' }}>Overlay Content</Text>
      </Box>
    </ZStack>
  );
}
```

## Box

A versatile container component with multiple variants and styling options.

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'card' \| 'surface'` | `'default'` | Box style variant |
| `padding` | `SpacingKey` | - | Internal padding |
| `margin` | `SpacingKey` | - | External margin |
| `backgroundColor` | `ColorKey` | - | Background color |
| `borderRadius` | `BorderRadiusKey` | - | Border radius |
| `borderWidth` | `number` | - | Border width |
| `borderColor` | `ColorKey` | - | Border color |
| `shadowOpacity` | `number` | - | Shadow opacity |
| `elevation` | `number` | - | Android elevation |
| `themed` | `boolean` | `true` | Apply theme styling |

### Variants

- **default**: Basic surface background

- **surface**: Background color variant

## Common Patterns

### Card Layout

```tsx
import { VStack, HStack, Box, Text, Center } from 'rnc-theme';

function ProfileCard() {
  return (
    <Box variant="card" margin="md">
      <VStack spacing="md">
        <HStack spacing="md" align="center">
          <Box
            width={60}
            height={60}
            backgroundColor="primary"
            borderRadius="full"
          >
            <Center flex={1}>
              <Text style={{ color: 'white', fontSize: 20, fontWeight: 'bold' }}>
                JD
              </Text>
            </Center>
          </Box>

          <VStack flex={1} spacing="xs">
            <Text style={{ fontSize: 18, fontWeight: 'bold' }}>John Doe</Text>
            <Text style={{ fontSize: 14, opacity: 0.7 }}>Software Developer</Text>
          </VStack>

          Passionate developer with 5+ years of experience in React Native
          and mobile app development.
        </Text>
      </VStack>
    </Box>
  );
}
```

## Best Practices

1. **Use semantic spacing**: Prefer theme spacing keys (`xs`, `sm`, `md`, `lg`, `xl`) over hardcoded values
2. **Leverage flex properties**: Use `flex` prop for responsive layouts
3. **Combine components**: Mix different layout components for complex interfaces
4. **Theme consistency**: Use `themed` prop and theme colors for consistent styling
5. **Performance**: Avoid deep nesting when possible, use flat structures with Grid for better performance

## TypeScript Support

All components are fully typed with TypeScript:

```tsx
import type { StackProps, BoxProps, GridProps } from 'rnc-theme';

// Custom component with proper typing
interface CustomLayoutProps extends StackProps {
  title: string;
  subtitle?: string;
}

function CustomLayout({ title, subtitle, ...stackProps }: CustomLayoutProps) {
  return (
    <VStack {...stackProps}>
      <Text style={{ fontSize: 18, fontWeight: 'bold' }}>{title}</Text>
      {subtitle && (
        <Text style={{ fontSize: 14, opacity: 0.7 }}>{subtitle}</Text>
      )}
    </VStack>
  );
}
```
