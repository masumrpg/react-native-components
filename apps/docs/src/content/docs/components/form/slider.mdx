The Slider component provides an intuitive way to select numeric values within a specified range. It includes both single-value and range slider variants with smooth gesture-based interactions.

## Features

- **Single & Range Sliders**: Support for both single value and range selection
- **Smooth Gestures**: Fluid pan gestures with spring animations
- **Multiple Variants**: Different color schemes for various use cases
- **Size Options**: Five different sizes from extra small to extra large
- **Step Support**: Configurable step increments for precise value selection
- **Live Labels**: Optional floating labels that show current values
- **Custom Formatting**: Customizable label formatters for different value displays
- **Accessibility**: Full accessibility support with proper semantics
- **Imperative API**: Programmatic control via refs

## Basic Usage

### Single Value Slider

```tsx
import { useState } from 'react';
import { Slider } from '@rnc/theme';
import { View, Text } from 'react-native';

export default function BasicSliderExample() {
  const [value, setValue] = useState(50);

  return (
    <View style={{ padding: 20 }}>
      <Text>Value: {value}</Text>
      <Slider
        initialValue={value}
        min={0}
        max={100}
        step={1}
        onValueChange={setValue}
        width={300}
      />
    </View>
  );
}
```

### Range Slider

```tsx
import { useState } from 'react';
```

## Imperative API

Control sliders programmatically using refs:

```tsx
import { useRef, useState } from 'react';
import { Slider, RangeSlider, SliderRef, RangeSliderRef } from '@rnc/theme';
import { View, Text, TouchableOpacity } from 'react-native';

export default function ImperativeSliderExample() {
  const sliderRef = useRef<SliderRef>(null);
  const rangeSliderRef = useRef<RangeSliderRef>(null);
  const [value, setValue] = useState(50);
  const [rangeValues, setRangeValues] = useState({ min: 20, max: 80 });

  const setRandomValue = () => {
    const randomValue = Math.floor(Math.random() * 100);
    sliderRef.current?.setValue(randomValue);
  };

  const setRandomRange = () => {
    const min = Math.floor(Math.random() * 40);
    const max = min + 20 + Math.floor(Math.random() * 40);
    rangeSliderRef.current?.setValues(min, max);
  };

  const getCurrentValues = () => {
    const currentValue = sliderRef.current?.getValue();
    const currentRange = rangeSliderRef.current?.getValues();
    console.log('Current slider value:', currentValue);
    console.log('Current range values:', currentRange);
  };

  return (
    <View style={{ gap: 20, padding: 20 }}>
      <View>
        <Text>Single Slider: {value}</Text>
        <Slider
          ref={sliderRef}
          initialValue={value}
          onValueChange={setValue}
          width={300}
        />
        <TouchableOpacity
          onPress={setRandomValue}
          style={{
            backgroundColor: '#007AFF',
            padding: 10,
            borderRadius: 5,
            marginTop: 10,
          }}
        >
          <Text style={{ color: 'white', textAlign: 'center' }}>
            Set Random Value
          </Text>
        </TouchableOpacity>
      </View>

      <View>
        <Text>Range Slider: {rangeValues.min} - {rangeValues.max}</Text>
        <RangeSlider
          ref={rangeSliderRef}
          initialMinValue={rangeValues.min}
          initialMaxValue={rangeValues.max}
          onValueChange={setRangeValues}
          width={300}
        />
        <TouchableOpacity
          onPress={setRandomRange}
          style={{
            backgroundColor: '#34C759',
            padding: 10,
            borderRadius: 5,
            marginTop: 10,
          }}
        >
          <Text style={{ color: 'white', textAlign: 'center' }}>
            Set Random Range
          </Text>
        </TouchableOpacity>
      </View>

      <TouchableOpacity
        onPress={getCurrentValues}
        style={{
          backgroundColor: '#FF9500',
          padding: 10,
          borderRadius: 5,
        }}
      >
        <Text style={{ color: 'white', textAlign: 'center' }}>
          Log Current Values
        </Text>
      </TouchableOpacity>
    </View>
  );
}
```

## Form Integration

Integrate with form libraries:

```tsx
import { useForm, Controller } from 'react-hook-form';
import { Slider, RangeSlider } from '@rnc/theme';
import { View, Text, TouchableOpacity } from 'react-native';

interface FormData {
  volume: number;
  priceRange: { min: number; max: number };
  quality: number;
}

export default function FormIntegrationExample() {
  const { control, handleSubmit, watch } = useForm<FormData>({
    defaultValues: {
      volume: 50,
      priceRange: { min: 100, max: 500 },
      quality: 80,
    },
  });

  const formData = watch();

  const onSubmit = (data: FormData) => {
    console.log('Form data:', data);
  };

  return (
    <View style={{ gap: 20, padding: 20 }}>
      <View>
        <Text>Volume: {formData.volume}</Text>
        <Controller
          control={control}
          name="volume"
          render={({ field: { onChange, value } }) => (
            <Slider
              initialValue={value}
              min={0}
              max={100}
              step={1}
              onValueChange={onChange}
              variant="info"
              width={300}
            />
          )}
        />
      </View>

      <View>
        <Text>
          Price Range: ${formData.priceRange.min} - ${formData.priceRange.max}
        </Text>
        <Controller
          control={control}
          name="priceRange"
          render={({ field: { onChange, value } }) => (
            <RangeSlider
              initialMinValue={value.min}
              initialMaxValue={value.max}
              min={0}
              max={1000}
              step={10}
              minDistance={50}
              onValueChange={onChange}
              variant="success"
              width={300}
            />
          )}
        />
      </View>

      <View>
        <Text>Quality: {formData.quality}</Text>
        <Controller
          control={control}
          name="quality"
          render={({ field: { onChange, value } }) => (
            <Slider
              initialValue={value}
              min={0}
              max={100}
              step={5}
              onValueChange={onChange}
              variant="warning"
              width={300}
            />
          )}
        />
      </View>

      <TouchableOpacity
        onPress={handleSubmit(onSubmit)}
        style={{
          backgroundColor: '#007AFF',
          padding: 15,
          borderRadius: 8,
        }}
      >
        <Text style={{ color: 'white', textAlign: 'center', fontWeight: '600' }}>
          Submit Form
        </Text>
      </TouchableOpacity>

      <Text style={{ fontSize: 12, color: '#666' }}>
        Current values: {JSON.stringify(formData, null, 2)}
      </Text>
    </View>
  );
}
```

## API Reference

### Slider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `initialValue` | `number` | `min` | Initial slider value |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `width` | `number` | `300` | Slider width |
| `height` | `number` | - | Custom height (overrides size) |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Slider size |
| `variant` | `'default' \| 'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error' \| 'info' \| 'destructive' \| 'outline' \| 'filled' \| 'ghost'` | `'default'` | Color variant |
| `disabled` | `boolean` | `false` | Whether slider is disabled |
| `trackColor` | `string` | - | Custom track color |
| `activeTrackColor` | `string` | - | Custom active track color |
| `thumbColor` | `string` | - | Custom thumb color |
| `thumbSize` | `number` | - | Custom thumb size |
| `showLabel` | `boolean` | `false` | Show floating value label |
| `labelFormatter` | `(value: number) => string` | `(value) => value.toString()` | Custom label formatter |
| `onValueChange` | `(value: number) => void` | - | Value change callback |
| `onSlidingStart` | `(value: number) => void` | - | Sliding start callback |
| `onSlidingComplete` | `(value: number) => void` | - | Sliding complete callback |
| `style` | `StyleProp<ViewStyle>` | - | Container styles |
| `trackStyle` | `StyleProp<ViewStyle>` | - | Track styles |
| `thumbStyle` | `StyleProp<ViewStyle>` | - | Thumb styles |
| `labelStyle` | `TextStyle` | - | Label text styles |

### RangeSlider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `initialMinValue` | `number` | `min` | Initial minimum value |
| `initialMaxValue` | `number` | `max` | Initial maximum value |
| `minDistance` | `number` | `0` | Minimum distance between thumbs |
| `showLabels` | `boolean` | `false` | Show floating value labels |
| `onValueChange` | `(values: { min: number; max: number }) => void` | - | Value change callback |
| `onSlidingStart` | `(values: { min: number; max: number }) => void` | - | Sliding start callback |
| `onSlidingComplete` | `(values: { min: number; max: number }) => void` | - | Sliding complete callback |

*All other props are inherited from the base Slider component.*

### SliderRef

| Method | Type | Description |
|--------|------|-------------|
| `setValue` | `(value: number) => void` | Set slider value programmatically |
| `getValue` | `() => number` | Get current slider value |

### RangeSliderRef

| Method | Type | Description |
|--------|------|-------------|
| `setValues` | `(min: number, max: number) => void` | Set range values programmatically |
| `getValues` | `() => { min: number; max: number }` | Get current range values |

## Best Practices

### Do's

- Use appropriate step values for the data type (integers vs decimals)
- Provide clear labels or descriptions for what the slider controls
- Use meaningful value ranges that make sense for your use case
- Consider showing current values alongside the slider
- Use appropriate variants to convey meaning (success for positive values, warning for thresholds)

### Don'ts

- Don't use sliders for precise value entry (use text inputs instead)
- Don't make sliders too narrow for comfortable interaction
- Avoid using sliders for boolean values (use switches instead)
- Don't use range sliders when a single value is sufficient
- Don't forget to handle edge cases in your value change handlers

### Accessibility

- Sliders automatically include proper accessibility labels
- Ensure sufficient color contrast for all visual states
- Test with screen readers to verify proper value announcements
- Consider providing alternative input methods for precise values
- Use descriptive labels that explain the slider's purpose

## Troubleshooting

### Common Issues

**Slider not responding to gestures**
- Ensure react-native-gesture-handler is properly installed and configured
- Check that the slider is not disabled
- Verify that parent views don't have gesture conflicts

**Values not updating correctly**
- Ensure `onValueChange` callback is provided
- Check that step values are appropriate for your range
- Verify that min/max values are set correctly

**Labels not showing**
- Ensure `showLabel` (single slider) or `showLabels` (range slider) is set to true
- Check that there's sufficient space above the slider for labels
- Verify that label styles are not hiding the text

**Performance issues with frequent updates**
- Consider debouncing value change callbacks for expensive operations
- Use `onSlidingComplete` instead of `onValueChange` for heavy operations
- Implement proper memoization for complex label formatters

**Range slider thumbs overlapping**
- Set an appropriate `minDistance` value
- Ensure the range between min and max values is sufficient
- Check that initial values respect the minimum distance
