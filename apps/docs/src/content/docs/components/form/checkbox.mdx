# Checkbox

A powerful, animated checkbox component with group support, multiple variants, custom shapes, and smooth spring animations. Built with React Native Reanimated for fluid user experiences.

## Features

  <Card title="✅ Group Support" icon="star">

## Basic Usage

```tsx
import { Checkbox } from 'rnc-theme';
import { useState } from 'react';

function BasicExample() {
  const [checked, setChecked] = useState(false);

  return (
    <View style={styles.container}>
      <Checkbox
        value="basic"
        checked={checked}
        onCheckedChange={setChecked}
      >
        <Checkbox.Label>Accept terms and conditions</Checkbox.Label>
      </Checkbox>
    </View>
  );
}
```

## API Reference

### Checkbox Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Unique identifier for the checkbox |
| `checked` | `boolean` | `false` | Controlled checked state |
| `onCheckedChange` | `(checked: boolean) => void` | - | Callback when checked state changes |
| `variant` | `ComponentVariant` | `'default'` | Visual style variant |
| `size` | `ComponentSize` | `'md'` | Checkbox size |
| `shape` | `'square' \| 'round'` | `'square'` | Checkbox shape |
| `disabled` | `boolean` | `false` | Disable the checkbox |
| `style` | `StyleProp<ViewStyle>` | - | Custom container styles |
| `children` | `React.ReactNode` | - | Checkbox content (label, indicator) |

### CheckboxGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string[]` | `[]` | Array of selected checkbox values |
| `onValueChange` | `(value: string[]) => void` | - | Callback when selection changes |
| `disabled` | `boolean` | `false` | Disable all checkboxes in group |
| `style` | `StyleProp<ViewStyle>` | - | Custom group container styles |
| `children` | `React.ReactNode` | - | Checkbox components |

### CheckboxIndicator Props

### Do's and Don'ts

<Tabs>
  <TabItem label="✅ Do's">
    ```tsx
    // ✅ Use clear, descriptive labels
    <Checkbox value="terms">
      <Checkbox.Label>I accept the Terms and Conditions</Checkbox.Label>
    </Checkbox>

    // ✅ Group related checkboxes
    <Checkbox.Group value={permissions} onValueChange={setPermissions}>
      <Checkbox value="read">Read Access</Checkbox>
      <Checkbox value="write">Write Access</Checkbox>
    </Checkbox.Group>

    // ✅ Use appropriate variants for context
    <Checkbox variant="error" value="delete">
      <Checkbox.Label>Permanently delete account</Checkbox.Label>
    </Checkbox>

    // ✅ Include accessibility support
    <Checkbox
      value="notifications"
      accessibilityLabel="Enable push notifications"
      accessibilityHint="Receive alerts when new messages arrive"
    >
      <Checkbox.Label>Push Notifications</Checkbox.Label>
    </Checkbox>
    ```
  </TabItem>

  <TabItem label="❌ Don'ts">
    ```tsx
    // ❌ Vague or unclear labels
    <Checkbox value="option1">
      <Checkbox.Label>Option 1</Checkbox.Label>
    </Checkbox>

    // ❌ Too many checkboxes without grouping
    <Checkbox value="a">A</Checkbox>
    <Checkbox value="b">B</Checkbox>
    <Checkbox value="c">C</Checkbox>
    // ... 20 more individual checkboxes

    // ❌ Inconsistent sizing
    <Checkbox size="xs">Small option</Checkbox>
    <Checkbox size="xl">Another option</Checkbox>

    // ❌ Missing accessibility
    <Checkbox value="important" /> // No label or accessibility info
    ```
  </TabItem>
</Tabs>

  When using large checkbox groups, consider implementing virtualization for lists with hundreds of items.
</Aside>

  The checkbox uses optimized spring animations. For better performance on older devices, you can disable animations by customizing the theme.
</Aside>

## Troubleshooting

### Common Issues

<Tabs>
  <TabItem label="State Management">
    **Problem:** Checkbox state not updating properly

    **Solutions:**
    ```tsx
    // Ensure proper controlled component pattern
    const [checked, setChecked] = useState(false);

    <Checkbox
      value="example"
      checked={checked}
      onCheckedChange={setChecked} // Make sure this is connected
    />

    // For groups, ensure value array is properly managed
    const [selectedItems, setSelectedItems] = useState<string[]>([]);

    <Checkbox.Group
      value={selectedItems}
      onValueChange={setSelectedItems} // This should update the array
    >
      {/* checkboxes */}
    </Checkbox.Group>
    ```
  </TabItem>

  <TabItem label="Animation Issues">
    **Problem:** Animations not working smoothly

    **Solutions:**
    ```tsx
    // Ensure react-native-reanimated is properly configured
    import 'react-native-reanimated/plugin';

    // Check for conflicting styles
    <Checkbox
      style={{
        // Avoid transform conflicts
        transform: undefined,
      }}
    />

    // For performance issues, consider reducing animation complexity
    // This is handled automatically by the component
    ```
  </TabItem>

  <TabItem label="Styling Issues">
    **Problem:** Custom styles not applying correctly

    **Solutions:**
    ```tsx
    // Use proper style props for different parts
    <Checkbox
      style={styles.checkboxContainer} // Container styles
      value="example"
    >
      <Checkbox.Indicator style={styles.indicator} />
      <Checkbox.Label style={styles.label}>Label</Checkbox.Label>
    </Checkbox>

    // Ensure theme compatibility
    const styles = useThemedStyles((theme) => ({
      checkboxContainer: {
        backgroundColor: theme.colors.surface,
      },
      label: {
        color: theme.colors.text,
      },
    }));
    ```
  </TabItem>
</Tabs>
