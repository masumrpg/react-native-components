# Button

A powerful, customizable button component that supports multiple variants, sizes, animations, loading states, and icons. Built with React Native Reanimated for smooth animations and full theme integration.

## Features

    ```tsx
      const buttonRef = useRef(null);
      const [step, setStep] = useState(0);

      const runSequence = async () => {
        setStep(1);
        buttonRef.current?.animate('pulse');

        setTimeout(() => {
          setStep(2);
          buttonRef.current?.animate('bounce');
        }, 500);

        setTimeout(() => {
          setStep(3);
          buttonRef.current?.animate('shake');
        }, 1000);

        setTimeout(() => {
          setStep(0);
        }, 1500);
      };

      return (
        <View style={styles.container}>
          <Button
            ref={buttonRef}
            variant="primary"
            onPress={runSequence}
            disabled={step > 0}
          >
            {step === 0 && 'Start Animation Sequence'}
            {step === 1 && 'Pulsing...'}
            {step === 2 && 'Bouncing...'}
            {step === 3 && 'Shaking...'}
          </Button>
        </View>
      );
    }
    ```

### Ref Usage

```tsx

function RefExample() {
  const buttonRef = useRef(null);
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `ComponentVariant` | `'default'` | Visual style variant |
| `size` | `ComponentSize` | `'md'` | Button size |
| `loading` | `boolean` | `false` | Show loading spinner |
| `disabled` | `boolean` | `false` | Disable button interaction |
| `fullWidth` | `boolean` | `false` | Make button full width |
| `icon` | `React.ReactNode` | - | Icon element |
| `iconPosition` | `'left' \| 'right' \| 'center'` | `'left'` | Icon position |
| `component` | `'pressable' \| 'touchable'` | `'pressable'` | Underlying component |
| `animationEnabled` | `boolean` | `true` | Enable press animations |
| `pressAnimationScale` | `number` | `0.95` | Scale factor for press animation |
| `springConfig` | `SpringConfig` | - | Custom spring animation config |
| `borderRadius` | `keyof Theme['components']['borderRadius']` | - | Custom border radius |
| `onPress` | `() => void` | - | Press handler |

### Variants

- `default` - Neutral button with surface background
- `primary` - Main call-to-action button
- `secondary` - Secondary action button
- `outline` - Bordered button with transparent background
- `filled` - Solid background button
- `ghost` - Minimal button with no background or border
- `success` - Success/positive action button
- `error` - Error/destructive action button
- `warning` - Warning/caution button
- `info` - Informational button

### Sizes

- `xs` - Extra small (28px height)
- `sm` - Small (32px height)
- `md` - Medium (40px height) - Default
- `lg` - Large (48px height)
- `xl` - Extra large (56px height)

### Ref Methods

| Method | Description |
|--------|-------------|
| `focus()` | Focus the button |
| `blur()` | Remove focus from button |
| `animate(type)` | Trigger custom animation |

### Animation Types

- `bounce` - Bounce scale animation
- `pulse` - Pulse scale animation
- `shake` - Horizontal shake animation
