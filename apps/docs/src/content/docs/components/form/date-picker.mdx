The Date Picker component provides an intuitive way to select dates through a calendar interface. It features smooth animations, multiple date formats, validation, and seamless integration with form libraries.

## Features

- **Calendar Interface**: Full calendar view with month/year navigation
- **Multiple Formats**: Support for various date formats (DD/MM/YYYY, MM/DD/YYYY, etc.)
- **Smooth Animations**: Fluid open/close animations with spring physics
- **Date Validation**: Min/max date constraints and validation
- **Marked Dates**: Highlight special dates with custom markers
- **Responsive Design**: Adapts to different screen sizes
- **Accessibility**: Full accessibility support with proper ARIA attributes
- **Form Integration**: Works seamlessly with form libraries
- **Customizable Styling**: Extensive theming and styling options
- **Portal Rendering**: Modal overlay for better UX

## Basic Usage

```tsx
import { DatePicker } from '@rnc/theme';
import { View } from 'react-native';
import { useState } from 'react';

export default function BasicDatePickerExample() {
  const [selectedDate, setSelectedDate] = useState<string>('');

  return (
    <View style={{ padding: 20 }}>
      <DatePicker
        label="Select Date"
        placeholder="Choose a date"
        value={selectedDate}
        onDateSelect={setSelectedDate}
      />
    </View>
  );
}
```

## Date Constraints

Set minimum and maximum date limits:

```tsx
import { DatePicker } from '@rnc/theme';
import { View } from 'react-native';
import { useState } from 'react';

export default function DateConstraintsExample() {
  const [birthDate, setBirthDate] = useState<string>('');
  const [appointmentDate, setAppointmentDate] = useState<string>('');
  const [eventDate, setEventDate] = useState<string>('');

  const today = new Date().toISOString().split('T')[0];
  const maxBirthDate = new Date(
    new Date().setFullYear(new Date().getFullYear() - 18)
  ).toISOString().split('T')[0];
  const minAppointmentDate = today;
  const maxAppointmentDate = new Date(
    new Date().setMonth(new Date().getMonth() + 3)
  ).toISOString().split('T')[0];

  return (
    <View style={{ gap: 20, padding: 20 }}>
      <DatePicker
        label="Birth Date (18+ only)"
        placeholder="Select your birth date"
        value={birthDate}
        onDateSelect={setBirthDate}
        maxDate={maxBirthDate}
        helperText="Must be 18 years or older"
      />

      <DatePicker
        label="Appointment Date"
        placeholder="Select appointment date"
        value={appointmentDate}
        onDateSelect={setAppointmentDate}
        minDate={minAppointmentDate}
        maxDate={maxAppointmentDate}
        helperText="Available for next 3 months"
      />

      <DatePicker
        label="Event Date (Future only)"
        placeholder="Select event date"
        value={eventDate}
        onDateSelect={setEventDate}
        minDate={today}
        helperText="Event must be in the future"
      />
    </View>
  );
}
```

## Form Integration

Integrate with form libraries like `react-hook-form`:

```tsx
import { useForm, Controller } from 'react-hook-form';
import { DatePicker } from '@rnc/theme';
import { Button } from '@rnc/theme';
import { View, Text, Alert } from 'react-native';

interface FormData {
  birthDate: string;
  startDate: string;
  endDate: string;
}

export default function FormIntegrationExample() {
  const {
    control,
    handleSubmit,
    watch,
    formState: { errors },
  } = useForm<FormData>({
    defaultValues: {
      birthDate: '',
      startDate: '',
      endDate: '',
    },
  });

  const startDate = watch('startDate');
  const today = new Date().toISOString().split('T')[0];

  const onSubmit = (data: FormData) => {
    Alert.alert('Form Submitted', JSON.stringify(data, null, 2));
  };

  return (
    <View style={{ gap: 20, padding: 20 }}>
      <Controller
        control={control}
        name="birthDate"
        rules={{
          required: 'Birth date is required',
          validate: (value) => {
            if (!value) return 'Birth date is required';
            const birthDate = new Date(value);
            const eighteenYearsAgo = new Date();
            eighteenYearsAgo.setFullYear(eighteenYearsAgo.getFullYear() - 18);
            return birthDate <= eighteenYearsAgo || 'Must be 18 years or older';
          },
        }}
        render={({ field: { onChange, value } }) => (
          <DatePicker
            label="Birth Date"
            placeholder="Select your birth date"
            value={value}
            onDateSelect={onChange}
            state={errors.birthDate ? 'error' : 'default'}
            error={errors.birthDate?.message}
            required
          />
        )}
      />

      <Controller
        control={control}
        name="startDate"
        rules={{
          required: 'Start date is required',
          validate: (value) => {
            if (!value) return 'Start date is required';
            return value >= today || 'Start date must be today or later';
          },
        }}
        render={({ field: { onChange, value } }) => (
          <DatePicker
            label="Start Date"
            placeholder="Select start date"
            value={value}
            onDateSelect={onChange}
            minDate={today}
            state={errors.startDate ? 'error' : 'default'}
            error={errors.startDate?.message}
            required
          />
        )}
      />

      <Controller
        control={control}
        name="endDate"
        rules={{
          required: 'End date is required',
          validate: (value) => {
            if (!value) return 'End date is required';
            if (!startDate) return 'Please select start date first';
            return value > startDate || 'End date must be after start date';
          },
        }}
        render={({ field: { onChange, value } }) => (
          <DatePicker
            label="End Date"
            placeholder="Select end date"
            value={value}
            onDateSelect={onChange}
            minDate={startDate || today}
            state={errors.endDate ? 'error' : 'default'}
            error={errors.endDate?.message}
            required
          />
        )}
      />

      <Button
        variant="primary"
        size="lg"
        onPress={handleSubmit(onSubmit)}
      >
        Submit Form
      </Button>
    </View>
  );
}
```

## Best Practices

### Do's

- Use appropriate date formats for your target audience
- Set reasonable min/max date constraints
- Provide clear labels and helper text
- Use marked dates to highlight important dates
- Consider timezone implications for your use case
- Test with different screen sizes

### Don'ts

- Don't use overly restrictive date ranges without explanation
- Avoid complex date validation without clear error messages
- Don't disable animations without good reason
- Avoid using the component for time-sensitive selections without proper validation
- Don't forget to handle edge cases like leap years

### Accessibility

- Date picker automatically provides proper ARIA labels
- Calendar navigation is keyboard accessible
- Screen readers announce selected dates
- Focus management is handled automatically
- High contrast mode is supported

## Troubleshooting

### Common Issues

**Date picker not opening**
- Ensure the component is not disabled
- Check if there are any overlay conflicts
- Verify that the Portal component is properly configured

**Wrong date format displayed**
- Check the `dateFormat` prop value
- Ensure the input date is in YYYY-MM-DD format
- Verify locale settings if using international formats

**Animation issues**
- Try reducing `animationDuration` for better performance
- Set `animated={false}` to disable animations if needed
- Check for conflicts with other animated components

**Calendar not showing marked dates**
- Ensure `markedDates` object uses YYYY-MM-DD format as keys
- Check that the marked date properties are correctly formatted
- Verify that dates are within the visible calendar range

**Performance issues**
- Reduce `animationDuration` for faster interactions
- Consider using `closeOnSelect={true}` for quicker selection
- Optimize marked dates object if it's very large
