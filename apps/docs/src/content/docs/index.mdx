## Tentang rnc-theme

**rnc-theme** adalah pustaka komponen UI React Native yang komprehensif dan sistem tema yang dirancang untuk mempercepat pengembangan aplikasi seluler Anda. Dengan fokus pada kustomisasi, kinerja, dan pengalaman pengembang yang luar biasa.

## 🚀 Instalasi Cepat

```bash
npm install rnc-theme
# atau
yarn add rnc-theme
```

## 💡 Penggunaan Dasar

```tsx
import React from 'react';
import { View } from 'react-native';
import { RNCProvider, Button, Typography } from 'rnc-theme';

const App = () => {
  return (
    <RNCProvider>
      <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
        <Typography variant="h1">Selamat Datang!</Typography>
        <Button onPress={() => console.log('Tombol ditekan!')}>
          <Typography color="white">Tekan Saya</Typography>
        </Button>
      </View>
    </RNCProvider>
  );
};

export default App;
```
