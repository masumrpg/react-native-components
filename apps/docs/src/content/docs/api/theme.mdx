Theme is core to the component system. It is used to style the components and provide a consistent look and feel to your app.

### ThemeMode

| Preset | Colors | Typography | Components |
|--------|--------|------------|------------|
| `system` | System default | Default | Default |
| `light` | Light theme colors | Default | Default |
| `dark` | Dark theme colors | Default | Default |

### Theme
Main theme object containing all styling configurations.

| Property | Type | Description |
|----------|------|-------------|
| `colors` | <a href='/api/theme#themecolors'>`ThemeColors`</a> | Color palette configuration |
| `fontSizes` | <a href='/api/theme#themefontsizes'>`ThemeFontSizes`</a> | Font size scale |
| `spacing` | <a href='/api/theme#themespacing'>`ThemeSpacing`</a> | Spacing scale |
| `components` | <a href='/api/theme#themecomponentsizes'>`ThemeComponentSizes`</a> | Component-specific sizing |
| `typography` | <a href='/api/theme#themetypography'>`ThemeTypography`</a> | Text style presets |

| `textSecondary` | `string` | Secondary text color |

| `accent` | `string` | Accent color |
| `destructive` | `string` | Destructive action color |

### ThemeTypography
Predefined text styles for consistent typography.

| Property | Type | Description |
|----------|------|-------------|
| `caption` | <a href='https://reactnative.dev/docs/text-style-props'>`TextStyle`</a> | Caption text style |
| `small` | <a href='https://reactnative.dev/docs/text-style-props'>`TextStyle`</a> | Small text style |
| `body` | <a href='https://reactnative.dev/docs/text-style-props'>`TextStyle`</a> | Body text style |
| `subtitle` | <a href='https://reactnative.dev/docs/text-style-props'>`TextStyle`</a> | Subtitle text style |
| `title` | <a href='https://reactnative.dev/docs/text-style-props'>`TextStyle`</a> | Title text style |
| `heading` | <a href='https://reactnative.dev/docs/text-style-props'>`TextStyle`</a> | Heading text style |

Each typography style contains:
- `fontSize`: Font size value
- `lineHeight`: Line height value
- `fontWeight`: Font weight value

### CustomThemeConfigFactory
Custom theme configuration factory function.

| Type | Return | Description |
|------|--------|-----------|
| `(isDark: boolean) => CustomThemeConfig` | <a href='/api/theme#customthemeconfig'>`CustomThemeConfig`</a> | Custom font size scale |

### CustomThemeConfig

CustomThemeConfig is a type alias that extends the base <a href='/api/theme#theme'>`Theme`</a> interface, allowing you to customize and override the default theme configuration while maintaining type safety and consistency across your application.
